# VECTOR 🤖
> "System online. Mainframe synchronized, Mr. Azadlari."

**Vector** is a minimalist, witty, and high-performance AI assistant built for the **MacBook M4**. Inspired by the JARVIS architecture, it serves as a personal co-pilot for software engineering students, specifically optimized for the CVUT ecosystem.

## ✨ Key Features
- **Voice Activated:** Real-time wake-word detection ("Vector").
- **Neural Engine:** Powered by **Groq (Llama 3.3 70B)** for lightning-fast, witty responses.
- **HUD Interface:** A sleek PyQt6-based Heads-Up Display for visual feedback.
- **CVUT Integrated:** Instant access to my Database Systems (DBS), SAP, and Math Analysis (MA1) schedules.
- **System Vitals:** Real-time monitoring of MacBook M4's CPU, RAM, and Battery health.
- **Gym Mode:** Integrated Spotify control to initiate bodybuilding sessions (Goal: 70kg).
- **Bolt Converter:** Quick currency conversion for Czech Crowns to Azerbaijan Manats.
- **Chess Stats:** Live tracking of rapid ratings and wins on Chess.com.

## 🛠 Tech Stack
- **Language:** Python 3.14
- **AI Brain:** Groq API (Llama 3)
- **GUI:** PyQt6
- **Speech:** Google Speech Recognition & macOS `say` engine
- **Vitals:** Psutil & Subprocess
